## Weather App
A modern and responsive weather web application built using **HTML, CSS, and JavaScript**.  
The app fetches real-time weather data using **WeatherAPI** and displays a 3-day forecast with a clean glassmorphism UI design.
---

## Live Demo
[Try Now](https://omarlokma.github.io/Weather_App/)

---
## Features
- Search weather by city name
- Get weather using current location (Geolocation API)
- 3-day weather forecast
- Wind speed & humidity details
- Modern glassmorphism UI
- Fully responsive design
---

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- WeatherAPI
- Font Awesome Icons
- Google Fonts (Poppins)
